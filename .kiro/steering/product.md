# Product Overview

PathGuide is an AI-powered educational guidance platform that helps students create personalized learning roadmaps and discover educational opportunities.

## Core Features

- **Roadmap Generation**: Conversational intake system that creates personalized learning paths based on user goals, background, and skill level
- **Industry Insights**: Real-time information about career trends, job markets, and industry developments across different regions
- **University Search**: Intelligent university and program recommendations with detailed admission guidance
- **Knowledge Testing**: Adaptive quiz generation and evaluation to assess user understanding

## User Experience

The application uses a dual-pane interface:
- **Chat Box**: Conversational interface for user interaction and clarifying questions
- **Result Box**: Visual display of generated roadmaps, university results, quiz boards, and insights

## AI Integration

Powered by OpenAI GPT-4o for:
- Natural language understanding and conversation
- Personalized content generation
- Educational content structuring
- Adaptive questioning and guidance
